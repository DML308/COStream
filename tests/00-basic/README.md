# basic.cos
>本测试用例用来测试基础文法的有效性
### 代码内容:
- `composite Main()`中
    - 将数组`a[8]`的值逐个赋值给流`S`
    - 再通过调用`composite B`来对 `S` 处理后,将结果赋值给流`P`
    - 最后在 `P` 的 `work`中逐个输出`P[0].x`
- `composite B`
    - 未对输入流`In`和输出流`Out`做任何特殊处理,仅仅做了赋值`Out[0].x = In[0].x;`,其余部分为测试代码.
### 预期效果:
#### 输入:
未使用命令行输入或文件输入, 使用了文件头部定义的`int a[8] = {1,2,3,4,5,6,7,8};`作为输入数据
#### 输出
将`a[8]`内容逐个输出, 即输出从1到8的8个数字
